Live Link

Project Overview

It's a responsive & intercative quiz app. User get's to choose category, difficulty, and format of questions on the home page. There after they get to proceed to quiz page where they can make their choices. When completed, answer can be checked by clicking on check answer. It displays result of quiz in visually appealing way. User can anytime toggle between menu page or play again.

This app is built using ReactJS. It uses OTDB api to fetch quiz questions data.
